# AI 智能面试系统 — 课设提交包

## 快速开始（2 步）

### 1. 配 API Key
编辑 `backend\.env`，填入 Claude API Key：
```
ANTHROPIC_API_KEY=你的key
```

### 2. 一键启动
双击 `start.bat`（Windows）自动完成：
- 安装依赖
- 启动后端 (localhost:8000)
- 启动前端 (localhost:3000)
- 打开浏览器

## 项目简介

双角色 AI 面试平台：求职者 AI 面试陪练 + HR 候选人管理。

### 求职者端
- 注册登录 → 设置目标岗位
- 上传简历 → AI 模拟技术面试
- 查看多维度评分报告和练习记录

### HR 端
- 创建岗位 → 生成邀请链接
- 候选人免登录面试
- 查看候选人面试报告和数据分析

## 技术架构
- 前端：Vue 3 + Nuxt 3（SPA）
- 后端：Python FastAPI
- AI：Claude API（多 Agent 架构）
- 数据库：SQLite

## 手动启动
```bash
# 终端1 — 后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 终端2 — 前端
cd frontend
npm install
npx nuxi dev --host 0.0.0.0
```

## 常见问题

**Q: 页面打不开？**
A: 确认后端 8000 端口和前端 3000 端口都启动了

**Q: 注册/登录失败？**
A: 检查后端是否正常运行，打开 http://localhost:8000/api/health 验证

**Q: AI 不回复？**
A: 检查 `backend\.env` 中 API Key 是否正确配置
